# ③ 단순화/유지보수성 (FSD 맥락)

변경된 코드가 **FSD 슬라이스/세그먼트 구조 안에서** 필요 이상으로 복잡하지 않은지 검증한다. 핵심 질문: **"FSD 경계를 지키면서 더 간단하게 작성할 수 없는가?"**

## FSD 슬라이스 복잡도 체크리스트

> 핵심 질문: **"이 슬라이스/세그먼트의 범위가 적절한가?"**

### 과도한 슬라이싱
- [ ] 슬라이스가 자체 폴더를 정당화하기에 너무 작은가? (예: model/api 없이 컴포넌트 하나만)
- [ ] 항상 함께 변경되는 여러 슬라이스가 있는가? → 하나로 합치기 검토
- [ ] 거의 빈 세그먼트가 있는가? (예: 한 줄짜리 훅만 있는 `model/`) → 별도 세그먼트 불필요할 수 있음

### 과소 슬라이싱
- [ ] 하나의 슬라이스가 관련 없는 여러 비즈니스 도메인을 처리하는가? → 분리 검토
- [ ] 슬라이스의 `model/`이 개념적으로 다른 관심사의 상태를 관리하는가? → 별도 슬라이스로 분리
- [ ] 슬라이스 내에서 한 파일이 300줄을 초과하는가? → 슬라이스의 책임이 과다한지 확인

### 크로스 슬라이스 중복
- [ ] 같은 로직이 여러 슬라이스에 중복되어 있는가? → 하위 레이어(entities 또는 shared)로 추출
- [ ] 여러 feature가 유사한 API 호출 패턴을 재구현하는가? → shared/api 또는 entities/*/api로 추출
- [ ] 유사한 UI 패턴이 슬라이스 간 복사되어 있는가? → shared/ui 또는 entities/*/ui로 추출

**예시**:

```
// ❌ 같은 검증 로직이 두 feature에 중복
features/checkout/lib/validateEmail.ts
features/auth/lib/validateEmail.ts

// ✅ shared로 추출
shared/lib/validation.ts  (또는 도메인 특화 시 entities/user/lib/validateEmail.ts)
```

```
// ❌ 과도한 슬라이싱 — 아주 작은 컴포넌트만 있는 슬라이스
features/logout/
├── ui/
│   └── LogoutButton.tsx  (10줄, auth.logout()만 호출)
└── index.ts

// ✅ features/auth/ui/LogoutButton.tsx에 포함하는 것이 적절
```

## 일반 복잡도 체크리스트

### 복잡도
- [ ] 함수/컴포넌트가 하나의 책임만 가지는가?
- [ ] 3단계 이상 중첩된 조건문/콜백이 있는가? → early return 또는 분리 가능
- [ ] 삼항 연산자가 중첩되어 있는가? → if/else 또는 별도 함수로 분리

### 과설계 (YAGNI)
- [ ] 현재 사용되지 않는 파라미터, 옵션, 설정이 추가되었는가?
- [ ] "나중에 필요할 것 같아서" 만든 추상화가 있는가?
- [ ] 한 곳에서만 사용되는 유틸/헬퍼 함수를 별도 파일로 분리했는가?
- [ ] 제네릭 타입이 과도하게 복잡한가? (실제 사용처 대비)

### 중복
- [ ] 같은 로직이 2곳 이상에서 반복되는가?
- [ ] 비슷하지만 약간 다른 컴포넌트/함수가 있는가? → 통합 가능 여부

### 원자적 변경
- [ ] PR에 관련 없는 변경이 섞여있지 않은가?
- [ ] 리팩토링과 기능 변경이 같은 PR에 있는가? → 분리 권장

### 가독성
- [ ] 변수/함수 이름이 의도를 명확히 표현하는가?
- [ ] 매직 넘버가 있는가? → 상수로 추출
- [ ] 주석 없이는 이해할 수 없는 코드가 있는가? → 코드 자체를 명확하게 개선

## 판정 기준

| 이슈 | 심각도 |
|------|--------|
| 같은 로직이 3개+ 슬라이스에 중복 | ❌ FAIL |
| 같은 로직이 2개 슬라이스에 중복 | ⚠️ WARNING |
| 과도한 슬라이싱 (사소한 슬라이스) | ⚠️ WARNING |
| 과소 슬라이싱 (여러 도메인 혼재) | ⚠️ WARNING |
| 3단계+ 중첩 조건문/콜백 | ⚠️ WARNING |
| YAGNI 위반 (미사용 추상화) | ⚠️ WARNING |
| 관련 없는 변경이 섞인 PR | ⚠️ WARNING |
| 매직 넘버 | ⚠️ WARNING |
| 300줄+ 파일 | ⚠️ WARNING |
